## 02. Setup

### Python virtual Environment

```powershell
# Enable powershell
PS> Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope Process -Force

# cd to root of future venv
PS> cd <targetdirectory for venv>

# list available python installations/versions
PS> py --list

# create venv for python version 3.11 using directory ./py311
PS> py -3.11 -m venv py311
```
