### Das Paket bauen (Build) ✔

Required: `pip install --upgrade build`

Build package: `python -m build`

Dadurch entsteht ein neuer Ordner namens `dist/`, der eine `.tar.gz`-Datei (Quellarchiv) und eine `.whl`-Datei (Wheel) enthält.

---

### Das Paket hochladen

Required: `pip install --upgrade twine`

Upload to TestPyPi: `python -m twine upload --repository testpypi dist/*`

Upload to PyPi: `python -m twine upload dist/*`
