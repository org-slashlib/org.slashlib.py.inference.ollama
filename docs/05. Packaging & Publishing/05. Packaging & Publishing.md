## 05. Packaging & Publishing

### Required: folder structure

```text
org.slashlib.py.agent/
├── src/org/slashlib/py/inference/ollama/  # sourcecode
│   ├── __init__.py
│   ├── adapter.py
│   └── result.py
│
├── tests/                                 # unittesting
├── README.md                              # project description for github/pypi etc.
├── LICENSE                                # license file
├── pyproject.json                         # JSON configuration file
└── pyproject.toml                         # project configuration file
```

---
### Required: `pyproject.toml`

```include toml
../../pyproject.toml
```

---
### Building the package

Required: `pip install --upgrade build`

Build package: `python -m build`

Dadurch entsteht ein neuer Ordner namens `dist/`, der eine `.tar.gz`-Datei (Quellarchiv) und eine `.whl`-Datei (Wheel) enthält.

---

### Uploading the package

Required: `pip install --upgrade twine`

Upload to TestPyPi: `python -m twine upload --repository testpypi dist/*`

Upload to PyPi: `python -m twine upload dist/*`
